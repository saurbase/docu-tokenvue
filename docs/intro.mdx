# What is TokenVue?

TokenVue is an AI Gateway and Infrastructure Control Platform that helps developers and organizations securely manage, route, monitor, and optimize AI traffic across multiple Large Language Model (LLM) providers and environments.

It acts as a unified control layer between applications and AI providers such as OpenAI, Anthropic, Gemini, DeepSeek, and open-source/self-hosted models.

---

## Why TokenVue Exists

Modern AI applications are becoming increasingly complex.

Teams often rely on:

* Multiple AI providers
* Different LLM models
* Open-source inference servers
* Hybrid AI infrastructure
* Custom routing logic
* Manual failover systems

This creates major challenges around:

* Reliability
* Security
* Observability
* Governance
* Cost optimization
* Vendor lock-in

TokenVue simplifies this complexity by providing one centralized platform to control and orchestrate AI traffic at production scale.

## OpenAI-Compatible API

TokenVue provides an OpenAI-compatible API layer, allowing developers to use existing OpenAI SDKs and workflows with minimal code changes.

```ts
import OpenAI from "openai";

const client = new OpenAI({
  apiKey: process.env.TOKENVUE_API_KEY,
  baseURL: "https://api.tokenvue.ai/v1",
});
```
